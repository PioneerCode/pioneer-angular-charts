Creating beautiful data visualizations using Pioneer Charts and Angular is both simple and intuitive. The following steps will help get you up and running in no time.

Introduction

Lets face it, creating rich data visualizations is hard! A lot of work needs to be put in up front to produce clean and compliant markup that reacts to data in meaningful way. Pioneer Charts sets out to solve these problems in a simple and intuitive way through:

  • Interaction with charts through a common and powerful contract.
  • Harnessing the power of d3 and the richness of SVG markup.
  • Building on top of Angular, a long-term supported and familiar framework.
  • Supplying a means to add styling overrides to an already beautiful default theme.
