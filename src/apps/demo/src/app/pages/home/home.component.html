Pioneer Charts

Data Visualizations in Angular, beautiful & simple.

Beautiful Charts

Data Visualizations don't need to be ugly. Pioneer Charts comes with a beautiful default theme of which is customizable to suit your needs.

Open Source

Pioneer Charts is developed and maintained in the open. Have a feature request or found a bug? Contributions are welcome!

Battle Tested Code

Pioneer Charts is built on top of Angular and backed by D3. Of which produces performant and consistent rendering across modern browser.
